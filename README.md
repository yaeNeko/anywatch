# AnyWatch : API REST

## Prérequis

-   [Node.js](https://nodejs.org/) installé

## Installation

1. **Cloner le repository**

    ```bash
    git clone https://gitlab.com/julesbsz/anywatch.git
    cd anywatch
    ```

2. **Installer les dépendances**

    ```bash
    npm install
    ```

3. **Créer un fichier .env dans la racine**

    ```bash
    DATABASE_URL=URL_DE_LA_BD
    ```

4. **Démarrer l'API**

    ```bash
    npm run dev
    ```
